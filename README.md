Turtle, quelle bibliothèque fascinante pour afficher des images à la vitesse de l'éclair. Malheureusement, vous ne trouvez pas que c'est un peu dépassé la 2D ? So let me introduce you to (comme u2) TURTLE 3D (i speak english very well). Tu prends un point obj en paramètres, grâce à ma technologie super innovante de parsing (ça lit le fichier). Le fichier 3D est dessiné avec Turtle, puis Pil enregistre l'image et le meilleur dans tout ça, c'est qu'il est possible de faire bouger et tourner le modèle pour créer une vidéo.
